## 屏 1 — 欢迎

### 目的
让用户知道这是个什么 App，建立第一印象。

### 进入条件
首次启动 App（settings.json 不存在 / onboardingComplete=false）。

### 区域

| 区域 | 控件 | 内容 / 行为 |
|---|---|---|
| 步骤标签 | text | `步骤 1 / 7`，11px gray-500，letterSpacing 0.6 |
| Logo wrap | frame 104×104 | accent 实色背景，圆角 24，shadow accent @ 20%；居中 |
| Logo icon | lucide `mic` 48×48 | 白色 |
| 品牌名 | text 32px | `LazyAudio`，gray-950，weight 700，letterSpacing -0.5 |
| Slogan | text 14px | `录下你电脑的声音 — 转成可搜索的文字 — 让 LLM 提炼要点`，gray-600 |
| 副标题 | text 11px | `本地优先 · macOS 14.2+ / Windows 10+ · 开源`，gray-400 |
| 底部主按钮 | button primary | `开始设置 →` |

### 状态变体
无（单状态）。

### 边界情况
- 用户关窗：弹"确定退出 LazyAudio 吗？下次启动会重新进入设置"

### Mockup 必画
- 默认态 ✅（已完成）

## 屏 2 — 隐私模式

### 目的
让用户在本地 / 云端之间二选一，决定后续走 4a 还是 4b 分支。

### 进入条件
屏 1 完成。

### 区域

| 区域 | 控件 | 内容 / 行为 |
|---|---|---|
| 头部 | vertical block | 步骤标签 + 标题 `选择隐私模式` + 副标题 `你的录音和转录数据如何处理 — 可以随时在设置里切换。` |
| 双卡片 | horizontal frame | 左右两个等宽卡片，gap 16 |

#### 卡片 — 本地（默认选中）
- 顶部 row：lucide `shield-check` 24px accent + 标题 `本地` 17px bold + 圆角 tag `推荐` (accent bg + 白字)
- 描述：`所有音频和转录留在你的机器上，离线可用。\n首次需下载约 270 MB 模型（SenseVoice）。`
- 选中态视觉：边框 2px accent + bg `accent @ 5%` + tag 可见

#### 卡片 — 云端
- 顶部 row：lucide `cloud` 24px gray-500 + 标题 `云端` 17px bold
- 描述：`用你的 OpenAI 兼容 API 转录，本机不存模型。\n速度依赖网络，需要 baseUrl 和 API key。`
- 未选中：边框 1px gray-200，hover bg gray-50

### 交互
- 点击卡片 = 切换选中
- 选中状态持久（用户在屏间前后退保留）
- 默认选中"本地"

### 状态变体
- 状态 A：本地选中（默认）
- 状态 B：云端选中

### 边界情况
- 网络不可达时云端卡片是否禁用？**不禁用** — 用户可能后续才联网。卡片下方加灰色小字 `需联网` 提示即可。

## 屏 4a — 模型下载（本地分支）

### 目的
下载 SenseVoice + Silero VAD，确保用户首次就能用本地转录。

### 进入条件
屏 2 选择"本地" + 屏 3 已授权（或跳过授权）。

### 区域

| 区域 | 控件 | 内容 / 行为 |
|---|---|---|
| 头部 | block | 步骤标签 + 标题 `下载本地转录模型` + 副标题 `首次需要下载约 270 MB。下载完成后所有转录都在本机进行。` |
| 模型卡片 | vertical frame | 见下 |
| 镜像信息 | small row | `从镜像 hf-mirror.com 下载` + `[切换镜像 ▾]` 链接 |
| 备选项 | row | `[跳过 — 暂时不能转录]` 灰色链接 ｜ `[切到云端]` 灰色链接 |

#### 模型卡片
- 顶部 row（horizontal space_between）：
  - 左：模型名 `SenseVoice-small (中英日韩粤, int8)` 15px medium + 副信息 `约 234 MB · MIT 协议` 11px gray
  - 右：状态文字 + 取消 / 暂停 icon button
- 进度条：高 8px，圆角 4，bg gray-200，fill accent；fill 宽度按 % 取
- 进度元信息行（horizontal）：
  - 百分比 `34%` 13px bold accent
  - 已下载 / 总 `79.5 MB / 234 MB` 11px gray
  - 速度 `2.4 MB/s` 11px gray
  - 剩余 `约 1 分 4 秒` 11px gray
- 文件列表（折叠面板，默认折叠）：
  - 点击展开：`model.int8.onnx 233.6 MB · 已下载 78%` / `tokens.txt 308 KB · ✓` / `silero_vad.onnx 1.8 MB · 等待中`

### 状态变体

| 状态 | 进度条 | 按钮 | 元信息 |
|---|---|---|---|
| 等待开始 | 0% | `[开始下载]` primary | `准备下载…` |
| 下载中（默认）| 进度更新 | `[⏸ 暂停]` icon | 百分比 / 速度 / ETA |
| 暂停 | 当前 % | `[▶ 继续]` primary | `已暂停` 灰字 |
| 校验中 | 100% | 隐藏 | `校验文件完整性 (SHA256)…` |
| 完成 | 100% accent | 隐藏 | ✓ `下载完成` success 色 |
| 失败 | 当前 % danger 色 | `[重试]` primary | 红字 `下载失败 — 网络连接超时` + `[复制错误]` |

### 镜像切换
- 默认根据用户区域：中国大陆 → hf-mirror，其他 → GitHub Releases
- `[切换镜像 ▾]` dropdown 选项：
  - `hf-mirror.com（国内推荐）`
  - `GitHub Releases（海外推荐）`
  - `HuggingFace 原站（兜底）`
- 切换时如已下载部分，提示"将丢失当前进度"

### 底部按钮行为
- `上一步`：暂停下载，回屏 3
- `下一步`：仅"完成"状态下 enabled

### 边界情况
- 磁盘空间不足：开始下载前预检查，不足时弹"需要至少 500 MB 可用空间"对话框 + 不开始
- 校验失败：自动重下 1 次，仍失败 → 失败态 + 错误信息 `文件校验失败 — 可能已损坏`
- 关闭 Onboarding 窗口：下载继续后台进行（提示"下载将在后台继续，可在设置中查看进度"）
- 用户在 90% 时按上一步：保留进度，不重置

### Mockup 必画
- 下载中（默认 34%）（**待画**）
- 完成（**待画**）
- 失败（**待画**）

## 屏 4b — API 配置（云端分支）

### 目的
让用户配置 OpenAI 兼容 API，测试通过后才能用云端转录。

### 进入条件
屏 2 选择"云端" + 屏 3 已授权（或跳过授权）。

### 区域

| 区域 | 控件 | 内容 / 行为 |
|---|---|---|
| 头部 | block | 步骤标签 + 标题 `配置 OpenAI 兼容 API` + 副标题 `LazyAudio 支持任意 OpenAI 兼容的转录 / chat API（OpenAI、Azure、Together、本地 Ollama 等）。` |
| 表单 | vertical frame | 见下 |
| 测试按钮 | button row | `[测试连接]` primary outline |
| 注脚 | info row | lucide `info` + `API key 用系统 keychain 加密存储，不会明文落盘。` |

#### 表单
3 个 labeled input：

| label | input 类型 | placeholder | 默认值 | 校验 |
|---|---|---|---|---|
| `Base URL` | text input | `https://api.openai.com/v1` | 空 | 必须以 `http(s)://` 开头 |
| `API Key` | password input + 显示切换 | `sk-...` | 空 | 非空 |
| `转录模型` | select + 自定义文本 | `gpt-4o-transcribe` 或自填 | 空 | 非空 |

可展开 "高级" 折叠区：
- `Chat 模型`（用于摘要）：select 同上，默认 `gpt-4o-mini`
- `请求超时`：number input，默认 60s
- `代理 URL`（可选）：text input

#### 测试连接行为
点击 `[测试连接]`：
- 按钮 → loading state `测试中…`
- 调用 transcribe endpoint 上传一段 1 秒静音 WAV（探针）
- 成功：右侧绿色 ✓ `连接成功，模型 {model_returned}` 12px success 色
- 失败：右侧红色 ✕ + 简短原因（如 `401 Unauthorized — API key 无效`），下方展开"技术详情"折叠区

### 状态变体

| 状态 | 测试按钮 | 状态提示 | 下一步 |
|---|---|---|---|
| 未填 / 未测试 | enabled | 无 | disabled `下一步（请先测试）` |
| 测试中 | `测试中…` disabled | 无 | disabled |
| 测试成功 | 显示 ✓ enabled | 绿字成功 | enabled |
| 测试失败 | enabled | 红字错误 + 技术详情链接 | disabled |

### 备选项
底部一行小字：`[跳过 — 暂时不能转录]` ｜ `[切到本地]` 灰色链接。

### 边界情况
- 用户改了表单字段：自动清除上次测试结果，下一步重新 disabled
- 测试中超时（>30s）：自动取消 + 失败态 `请求超时`
- 用户填了部分字段就 `下一步`：阻止 + toast `请填完所有字段并测试连接`

### Mockup 必画
- 未填 / 未测试（**待画**）
- 测试成功 ✓（**待画**）
- 测试失败 ✕（**待画**）

## 屏 6 — 合规提示

### 目的
一次性告知"录制对方音频前需告知对方"，避免用户日后陷入麻烦。

### 进入条件
屏 5 完成。

### 区域

| 区域 | 控件 | 内容 / 行为 |
|---|---|---|
| 头部 | block | 步骤标签 + 标题 `录音合规提示` |
| 主内容 | vertical card | 见下 |
| 复选框 | row | `[ ] 不再提示此信息`（默认未勾选）|
| 主按钮 | primary | `我已知晓` |

#### 主内容卡
- 顶部 icon: lucide `scale` 32px gray-700
- 一段正文（13px gray-700 line-height 1.7）：

```
LazyAudio 让你能录下电脑系统音和麦克风。在大多数国家和地区，
你录自己的声音、自己的会议没有问题，但有些地方对录制他人的声音
有更严格的规定（如美国部分州的 two-party consent 法律）。

在录制对方音频之前，请遵守当地法律，并按需告知对方。

LazyAudio 不会上传你的录音 — 但你与他人分享录音 / 转录时，
请同样注意对方的同意权。
```

### 行为
- 仅"我已知晓"按钮可进入下一步
- 勾选"不再提示"：写入 settings.json `complianceReminderHidden = true`；不勾选则录音前还会零星提示（v0.2 再做，v0.1 仅 onboarding 一次）

## 屏 7 — 完成

### 目的
让用户感到"准备好了，可以开始用了"，并立即引导第一次录音。

### 进入条件
屏 6 完成。

### 区域

| 区域 | 控件 | 内容 / 行为 |
|---|---|---|
| 大成功 icon | block | 居中 success 圆形 64px + ✓ icon |
| 标题 | text 24px bold | `准备好了` |
| 副标题 | text 13px | `按 ⌘⇧R 开始你的第一次录音，或点下面的按钮。` |
| Tip 卡 | small card | lucide `lightbulb` + `录音中点关闭窗口不会停止录音，LazyAudio 会留在菜单栏。` |
| 主按钮 | primary large | `🎤 开始第一次录音`（点击 = 触发录音浮窗 + 关闭 onboarding） |
| 次要链接 | text link | `稍后 — 先打开主窗口看看` |

### 行为
- `开始第一次录音` 点击：
  1. 关闭 onboarding 窗口
  2. 主窗口启动到后台（最小化到菜单栏）
  3. 立即弹出录音前浮窗
- `稍后`：仅关闭 onboarding，打开主窗口（空状态）
- 写入 settings.json `onboardingComplete = true`，下次启动不再走 onboarding

## 跨屏一致性约束

| 项 | 规则 |
|---|---|
| 窗口最大 | 高度固定 640，宽度 880，**不允许 resize** |
| 内边距 | 标准 padding `[40, 56]` |
| 头部 gap | 步骤标签 ↓ 8 ↓ 标题 ↓ 8 ↓ 副标题 |
| 主体到底部 footer gap | 32 |
| 主按钮 | accent primary, height 36-40, padding 18 |
| 步骤标签 | 11px 700 letterSpacing 0.6 gray-400，固定居中 |
| 上一步 | 屏 1 不显示，屏 2–7 都显示 |
| 关闭按钮 | 标题栏 × ，弹确认 dialog |
| 键盘 | Enter = 下一步 / 主 CTA；Esc = 关闭并弹确认；Tab 焦点循环 |
